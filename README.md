# Noomo Inspired Visual Site

An original high-end dynamic visual website inspired by the motion language of immersive WebGL showcase sites.

## Features

- Animated loading progress
- Scroll-driven canvas background
- Dynamic progress rail and bottom progress bar
- Responsive desktop and mobile layouts
- GitHub Pages deployment through Actions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
